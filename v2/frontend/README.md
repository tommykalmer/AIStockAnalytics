# Frontend v2 – AIStockAnalytics

Här utvecklas den nya frontend-delen i Next.js 14+, med:

- Landningssida (återanvänds delvis från v1)
- Inloggning / registrering
- Butikssida för licenser och engångsköp
- Dashboard för licenskunder
- Engångsköpsflöde utan konto

Målet är att ge en tydlig, mobilvänlig och lågtrösklig upplevelse.
